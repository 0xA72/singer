# Singer

Реализация простого HTTP(S) proxy сервера, предназначенного для перехвата, анализа и частичной модификации трафика между клиентом и целевыми веб‑сервисами.

### Ключевые возможности
- использование любых сторонних DNS серверов без изменения глобальных параметров локальной сети;
- перехват запросов на соединение и предоставление информации для последующего анализа;
- модификация в части фрагментации ClientHello запроса на установление защищенного соединения в соответсвии с пунктом 6.2.1 RFC 5246 в исследовательских целях.

### Побочные эффекты
- восстановление работоспособности интернет ресурсов после их деградации.

### Запуск
```
you@computer ~ % node singer
```
